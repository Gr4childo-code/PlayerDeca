## Приступая к работе

Сначала запустите сервер разработки:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## API

### Audios
```bash
[domain]/api/audios
```

### Events
```bash
[domain]/api/events
```

### Playlists
```bash
[domain]/api/playlists
```

### Styles
```bash
[domain]/api/styles
```
